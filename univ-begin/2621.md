📌 코드 흐름 분석

int v[5], s, i, score; 
char c[4]; 
int fl;

	•	v[5] : 5장의 카드 숫자를 저장 (예: 10, 11, 12 …)
	•	c[4] : 카드의 무늬(suit)를 저장 (예: H, D, S, C)
	•	fl : 모든 카드의 무늬가 같은지 여부 확인용

⸻

1. 입력 처리

for(i=0;i<5;i++) { 
    scanf("%s%d",c,v+i); 
    if(i==0) 
        fl=c[0]; 
    else 
        fl=(fl==c[0])?fl:0; 
}

	•	카드 5장 입력받음.
입력 형식: "무늬 숫자" → 예: H 10, S 11 이런 식
	•	fl :
	•	첫 번째 카드에서 무늬를 기록 (fl=c[0])
	•	이후 카드들이 전부 같은 무늬인지 확인
	•	다르면 fl=0으로 설정됨

👉 결과: fl != 0 이면 플러시(Flush, 무늬 동일), 아니면 일반

⸻

2. 정렬

std::sort(v,v+5);

	•	카드 숫자를 오름차순으로 정렬 (족보 판정에 편함)

⸻

3. 스트레이트(연속 여부) 체크

for(i=1;i<5;i++) if(v[i]!=v[0]+i) break; 
bool st=i==5;

	•	맨 앞 숫자 v[0] 기준으로 연속된 숫자인지 확인
	•	다 연속이면 st=true → 스트레이트

⸻

4. 점수 계산 (족보별 분기)

여기가 핵심!
큰 족보부터 작은 족보 순서로 조건 검사 후 score 결정.

⸻

(1) Straight Flush

if(st&&fl) score=900+v[4];

	•	무늬 같고(fl), 숫자 연속(st)
	•	가장 큰 족보 → 900+가장 높은 숫자

⸻

(2) Four of a Kind (포카드)

else if(v[1]==v[2]&&v[2]==v[3]&&(v[0]==v[1]||v[3]==v[4])) 
    score=800+v[1];

	•	중간 3장이 모두 같고, 양쪽 하나가 붙은 형태
	•	점수 800+해당 숫자

⸻

(3) Full House (풀하우스)

else if(v[0]==v[1]&&v[1]==v[2]&&v[3]==v[4]) 
    score=700+v[0]*10+v[4];
else if(v[0]==v[1]&&v[2]==v[3]&&v[3]==v[4]) 
    score=700+v[4]*10+v[0];

	•	AAA BB 또는 AA BBB 형태
	•	점수 700 + (트리플 숫자*10 + 페어 숫자)

⸻

(4) Flush

else if(fl) score=600+v[4];

	•	무늬 같음
	•	600+가장 큰 숫자

⸻

(5) Straight

else if(st) score=500+v[4];

	•	연속됨
	•	500+가장 큰 숫자

⸻

(6) Three of a Kind (트리플)

else if(v[0]==v[1]&&v[1]==v[2]) score=400+v[0];
else if(v[1]==v[2]&&v[2]==v[3]) score=400+v[1];
else if(v[2]==v[3]&&v[3]==v[4]) score=400+v[2];


⸻

(7) Two Pairs (투페어)

else if(v[0]==v[1]&&v[2]==v[3]) score=300+v[2]*10+v[0];
else if(v[0]==v[1]&&v[3]==v[4]) score=300+v[3]*10+v[0];
else if(v[1]==v[2]&&v[3]==v[4]) score=300+v[3]*10+v[1];


⸻

(8) One Pair (원페어)

else if(v[0]==v[1]) score=200+v[0];
else if(v[1]==v[2]) score=200+v[1];
else if(v[2]==v[3]) score=200+v[2];
else if(v[3]==v[4]) score=200+v[3];


⸻

(9) High Card (탑)

else score=100+v[4];

	•	아무 족보 없으면 가장 큰 카드 숫자

⸻

5. 출력

printf("%d\n",score);

📌 정리

이 코드는 포커 족보 판정기예요.
	•	입력: 카드 5장 (무늬+숫자)
	•	처리:
	1.	숫자 정렬
	2.	같은 무늬 확인 (Flush)
	3.	연속 여부 확인 (Straight)
	4.	족보 큰 순서대로 if문 검사
	•	출력: 족보에 따른 점수
